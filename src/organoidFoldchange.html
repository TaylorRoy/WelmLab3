Organoid Drug Screen Foldchange Tool

This tool was made to process EnVision plate readout files for the organoid drug screen project.

Tool calculates average of four reads per sample, then divides by average background for each plate- creating foldchange data for the Organoid Drug Screen which can be put into Graph Prism to produce heat map plots.

Important Rules

1.) Excel files must be saved to .csv

1.) Must have at least 5 plate reads, and be in groups of 5 (ex. 5, 10, 15, etc.)

1.) Read mush have 6 blanks in column 24.

1.) Drug file must have 50 drugs.

2.) The word "first" must be placed in front of the first desired read value for each plate.

3.) The word "last" must be placed behind the last desired read value for each plate.
